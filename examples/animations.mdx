## Wave Directions

Control the direction of wave animations using the `direction` prop. This affects how the animation propagates across the grid.

### Horizontal Waves

<CodeGroup>

```tsx Left to Right
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="all" 
      direction="ltr" 
      variant="purple" 
    />
  );
}
```

```tsx Right to Left
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="all" 
      direction="rtl" 
      variant="cyan" 
    />
  );
}
```

</CodeGroup>

## Animation Duration

Control the speed of animations using the `duration` prop (in seconds):

<CodeGroup>

```tsx Fast Animation (0.4s)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="wave-lr" 
      duration={0.4} 
      variant="sky" 
    />
  );
}
```

```tsx Normal Animation (0.7s)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="wave-lr" 
      duration={0.7} 
      variant="lime" 
    />
  );
}
```

```tsx Slow Animation (1.5s)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="wave-lr" 
      duration={1.5} 
      variant="orange" 
    />
  );
}
```

```tsx Very Slow Animation (3s)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="wave-lr" 
      duration={3} 
      variant="pink" 
    />
  );
}
```

</CodeGroup>

## Sequence Animations

Some patterns use sequence animations instead of waves, cycling through multiple frames:

<CodeGroup>

```tsx Corners Only (Sequence)

export default function App() {
  return (
    <Lumidot 
      pattern="corners-only" 
      variant="blue" 
    />
  );
}
```

```tsx Plus Hollow (Sequence)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      pattern="plus-hollow" 
      variant="fuchsia" 
    />
  );
}
```

```tsx Spiral (Sequence)
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={5} 
      cols={5} 
      pattern="spiral" 
      variant="purple" 
    />
  );
}
```

```tsx Spiral Reversed

      rows={5} 
      cols={5} 
      pattern="spiral"

```

</CodeGroup>

## Combining Direction with Patterns

<CodeGroup>

## Duration with Different Grid Sizes

Larger grids may need longer durations for smooth wave effects:

<CodeGroup>

```tsx Small Grid, Fast
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={3} 
      cols={3} 
      pattern="wave-lr" 
      duration={0.5} 
      variant="violet" 
    />
  );
}
```

```tsx Large Grid, Moderate
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={7} 
      cols={7} 
      pattern="wave-lr" 
      duration={1.2} 
      variant="sky" 
    />
  );
}
```

```tsx Extra Large Grid, Slow
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={10} 
      cols={10} 
      pattern="all" 
      duration={2} 
      variant="lime" 
      scale={1.2}
    />
  );
}
```

</CodeGroup>
