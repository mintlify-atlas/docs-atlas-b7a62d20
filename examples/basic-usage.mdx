## Default Loading Animation

The simplest usage with default settings:

```tsx
import { Lumidot } from 'lumidot';

export default function App() {
  return <Lumidot />;
}
```

This creates a 3x3 grid with blue dots and the default 'all' pattern.

## Different Patterns

Lumidot comes with 36 built-in patterns. Here are some common ones:

<CodeGroup>


## Customizing Appearance

Adjust glow intensity and scale:

<CodeGroup>

```tsx No Glow
import { Lumidot } from 'lumidot';

export default function App() {
  return <Lumidot glow={0} variant="slate" />;
}
```

```tsx Intense Glow
import { Lumidot } from 'lumidot';

export default function App() {
  return <Lumidot glow={20} variant="fuchsia" />;
}
```

```tsx Large Scale
import { Lumidot } from 'lumidot';

export default function App() {
  return <Lumidot scale={2} variant="cyan" />;
}
```

```tsx Small Scale
import { Lumidot } from 'lumidot';

export default function App() {
  return <Lumidot scale={0.5} variant="emerald" />;
}
```

</CodeGroup>
