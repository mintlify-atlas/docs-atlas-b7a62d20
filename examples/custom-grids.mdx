## Grid Dimensions

By default, Lumidot creates a 3x3 grid. You can customize the grid dimensions using the `rows` and `cols` props.

## Wide Grids

Create wide loading indicators with more columns than rows:

<CodeGroup>

```tsx 1x5 Grid
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={1} 
      cols={5} 
      pattern="wave-lr" 
      variant="purple" 
    />
  );
}
```

```tsx 2x8 Grid
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={2} 
      cols={8} 
      pattern="wave-lr" 
      variant="cyan" 
    />
  );
}
```

```tsx 3x7 Grid with Frame
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={3} 
      cols={7} 
      pattern="frame" 
      variant="emerald" 
    />
  );
}
```

</CodeGroup>

## Combining Grid Size with Scale

Adjust both dimensions and visual scale for precise sizing:

<CodeGroup>

```tsx Large Grid, Small Dots
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={8} 
      cols={8} 
      pattern="frame" 
      variant="cyan" 
      scale={0.8}
    />
  );
}
```

```tsx Small Grid, Large Dots
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={2} 
      cols={2} 
      pattern="corners-sync" 
      variant="rose" 
      scale={2.5}
    />
  );
}
```

```tsx Wide Grid, Scaled Up
import { Lumidot } from 'lumidot';

export default function App() {
  return (
    <Lumidot 
      rows={2} 
      cols={10} 
      pattern="wave-lr" 
      variant="amber" 
      scale={1.8}
    />
  );
}
```

</CodeGroup>
